# FoxiaModInstaller

This tool is made for Foxia to quickly install simple voice chat and other client side mods quickly.

Mods used:

Simple Voice Chat; 
Sodium; 
Lithium; 
Starlight; 
Exordium; 
Entity Culling; 
ModernFix; 
Krypton; 
Nvidium; 
Mod Menu; 
FerriteCore; 
LazyDFU;
